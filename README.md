# 线下活动现场大屏签到抽奖系统

一个纯前端实现的线下活动签到抽奖系统，无需后端、无需联网，本地打开即用。

## 🎯 功能特点

### 视觉效果
- 深色科技感主题，适合舞台大屏投影
- 全屏动态粒子背景，带连线效果
- 渐变流光文字和霓虹光效
- 响应式布局，适配各种屏幕尺寸

### 签到功能
- 输入姓名和公司完成签到
- 自动检测重复签到
- 签到成功飞入动画
- 实时显示签到人员列表
- 支持一键清空签到名单
- 本地数据持久化

### 抽奖功能
- 支持抽取 1/3/5/10 名获奖者
- 3秒倒计时动画
- 名字快速滚动 + 高亮闪烁
- 中奖时全屏礼花特效
- 自动排除已中奖人员
- 金色中奖卡片展示
- 支持重置抽奖状态
